## Installation

Download the latest release for your platform from the [GitHub releases page](https://github.com/dylanbr0wn/shsh/releases).

| Platform | Download |
|----------|----------|
| macOS    | `.dmg`   |
| Windows  | `.exe`   |
| Linux    | `.AppImage` |

## First Connection

1. Open shsh
2. Click **Quick Connect** on the welcome screen
3. Enter your host details (hostname, username, port)
4. Click **Connect**

You're in! The terminal session opens in a new workspace tab.

## Saving Hosts

To save a host for future use:

1. Click **Add Host** on the welcome screen (or **File → Add Saved Host**)
2. Fill in the connection details
3. Optionally assign a color and group
4. Click **Save**

Saved hosts appear in the sidebar for one-click connections.
