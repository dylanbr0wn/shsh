## Workspaces

Each connection opens in its own workspace tab. You can have multiple workspaces open simultaneously and switch between them using the tab bar.

## Split Panes

Within a workspace, you can split the view into multiple panes:

- **Vertical split** — divide the workspace side by side
- **Horizontal split** — stack panes top and bottom

Each pane can hold a terminal session, SFTP browser, or port forwarding view — all sharing the same SSH connection.

## Managing Sessions

- **Disconnect** — close the current session from the Session menu or by closing the workspace tab
- **Disconnect All** — close all active sessions at once via **Session → Disconnect All**
