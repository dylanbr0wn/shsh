## Dark and Light Mode

shsh supports dark and light themes. Toggle between them in **Settings**.

## Terminal Profiles

Customize terminal appearance per-host or globally:

1. Go to **File → Terminal Profiles**
2. Create or edit a profile
3. Adjust font family, font size, and color scheme
4. Assign the profile to specific hosts or set it as the default
