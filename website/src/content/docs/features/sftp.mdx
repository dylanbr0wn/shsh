## File Browser

Open an SFTP pane from any active connection to browse the remote file system. The browser supports:

- **Breadcrumb navigation** — click any segment to jump to that directory
- **File operations** — create directories, rename, and delete files
- **Drag and drop** — drop local files onto the browser to upload

## Uploading and Downloading

- **Upload** — drag files into the SFTP pane or use the upload button in the toolbar
- **Download** — select a file and click the download button to save it locally
