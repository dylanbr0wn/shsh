---
title: Port Forwarding
description: Set up local and remote port forwards over SSH
---

## Adding a Port Forward

1. Open a connected session
2. Go to **Session → Add Port Forward** (or use the port forwarding pane)
3. Enter the local port and remote host/port
4. Click **Add**

The forward starts immediately. Active forwards are listed in the port forwarding pane with their local and remote addresses.

## Removing a Forward

Click the remove button next to any active forward to stop it.
