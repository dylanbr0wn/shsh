## Importing from ~/.ssh/config

shsh can read your existing SSH config file:

1. Go to **File → Import SSH Config**
2. Review the detected hosts
3. Select which hosts to import
4. Click **Import**

Imported hosts appear in the sidebar alongside manually added hosts.

## Exporting Hosts

To export your saved hosts:

1. Go to **File → Export Hosts**
2. Choose the export format
3. Save the file
